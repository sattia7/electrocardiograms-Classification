## ğŸ«€ What Is an ECG?

An **Electrocardiogram (ECG)** measures the electrical signals generated by the heart as it beats.

An ECG signal is:

- A time-series signal
- A 1D numerical sequence
- Represented as voltage values over time

Each heartbeat produces a characteristic waveform:

- **P wave**
- **QRS complex**
- **T wave**

Abnormal heartbeats modify this waveform shape.
